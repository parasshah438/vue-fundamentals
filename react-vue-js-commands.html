React + Laravel Setup

Choose your preferred method to integrate React with Laravel.

Laravel Breeze (React)

Starter Kit — Authentication Scaffolding

Beginner

The simplest way to get started with React in Laravel. Breeze provides minimal authentication views using Inertia.js and React.

  1. 1Install Breeze via Composer as a dev dependency
  2. 2Scaffold React + Inertia authentication
  3. 3Install NPM dependencies and compile assets
  4. 4Run database migrations
1# Install Breeze package
2$ composer require laravel/breeze --dev
3
4# Scaffold React authentication
5$ php artisan breeze:install react
6
7# Install dependencies & build
8$ npm install
9$ npm run dev
10
11# Run migrations
12$ php artisan migrate
Laravel Jetstream (React)

Advanced Kit — Teams, 2FA, API Tokens

Advanced

Full-featured starter kit with team management, two-factor authentication, session management, API support via Sanctum, and profile management.

  1. 1Install Jetstream via Composer
  2. 2Scaffold Inertia.js stack with React
  3. 3Install frontend dependencies and build assets
  4. 4Run database migrations
1# Install Jetstream
2$ composer require laravel/jetstream
3
4# Scaffold with Inertia + React
5$ php artisan jetstream:install inertia --react
6
7# Install & build
8$ npm install
9$ npm run dev
10
11# Run migrations
12$ php artisan migrate
Laravel + React via Vite

Pure React — Custom Setup with Vite

Intermediate

Set up React manually with Vite for full control over your frontend. Best for custom apps without starter kit scaffolding.

1# Install React core packages
2$ npm install react react-dom
3
4# Install Vite React plugin (dev dependency)
5$ npm install -D @vitejs/plugin-react
6
7# (Optional) Install Babel preset + Vite plugin
8$ npm install --save-dev @babel/preset-react @vitejs/plugin-react vite
Laravel + React via Inertia.js

Modern Monolith — SPA without API

Intermediate

Inertia.js lets you build SPAs using classic server-side routing. No need to build a separate API — works seamlessly with Laravel controllers.

1# Install server-side adapter
2$ composer require inertiajs/inertia-laravel
3
4# Install React client-side adapter
5$ npm install @inertiajs/react
Laravel API + Separate React App

Decoupled — API Backend + React Frontend

Advanced

Build Laravel as a pure REST API and create a completely separate React frontend. Best for microservices architecture and team separation.

1# Create a separate React frontend app
2$ npx create-react-app frontend
3
4# Or use Vite (recommended)
5$ npm create vite@latest frontend -- --template react

Laravel + Vue via Vite

Pure Vue — Custom Setup with Vite

Intermediate

Set up Vue 3 manually with Vite for full control. Laravel ships with Vite support by default — just add Vue!

1# Install Vue 3
2$ npm install vue@latest
3
4# Install Vite Vue plugin
5$ npm install -D @vitejs/plugin-vue
6
7# (Optional) Vue SFC compiler
8$ npm install -D vue-loader @vue/compiler-sfc
